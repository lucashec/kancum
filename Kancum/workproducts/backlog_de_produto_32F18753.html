Description
Main DescriptionO backlog do produto é uma lista de recursos novos, aprimoramentos, atualizações de segurança, tarefas ou requisitos de trabalho necessários para criar um produto. Ele é compilado a partir de fontes de informações como suporte ao cliente, análise da concorrência, demandas de mercado e análise geral de negócios.
