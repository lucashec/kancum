      Main Description

      Todos os dias da Sprint, o Scrum Team realiza reuniões diárias. Essas reuniões geralmente são realizadas no mesmo local e horário todos os dias. O ideal é que a Reuniões Diária seja realizada pela manhã, uma vez que ela apoia a determinar as atividades do dia.

      Todos os membros do time devem participar. Outras pessoas também podem participar para contribuir sobre determinado assunto.

      A reunião não tem o objetivo de solucionar problemas ou dúvidas técnicas. Para isso, melhor se aplica uma reunião indivizualizada com a parte interessada.

      Durente a reunião, cada membro responde as seguintes perguntas:


      1. O que você fez ontem?
      2. O que você vai fazer hoje?
      3. Existe algum impedimento?
